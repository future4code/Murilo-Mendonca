## Projeto Cookenu - Murilo Mendonca

Projeto elaborado para conclusão da semana 18, do módulo de backend.

## Requisitos básicos:
* Todos os endpoints, com exceção do Signup e Login, devem exigir autenticação.

- Signup
    Método: POST
    Path: /signup

- Login
    Método: POST
    Path: /login

- Pegar próprio perfil
    Método: GET
    Path: /user/profile 

- Pegar próprio perfil
    Método: GET
    Path: /user/:id

- Criar receita 
    Método: POST
    Path: /recipe

- Pegar receita
    Método: GET
    Path: /recipe/:id
